  Find the longest substring that contains all unique values

  <pre>

  <h3>Stepping through the algorithm:</h3>
  <ol>
    <li>
      Lets initialize some variables, we want longest, seen and start
    </li>
    <li>
      Iterate through the string as you would ltr 
    </li>
    <li>
      Create a pointer to the character
    </li>
    <li>
      Lets check the seen object we instiated to see if an a character was indeed seen
    </li>
    <li>
      If we seen the character lets set the start value to be the largest of either the seen 
      count of that character. or a previous, or 0 if we are just starting.
    </li>
    <li>
      Now we can determine the longest value, we math.max to get the largest value , either the longest we have seen at this time
      or the current idx that we calculate 
    </li>
    <li>
      Finally lets update the seen Object with the current Index
    </li>
    <li>
      The end of this iteration, we return the longest that we've found. thats it. 
    </li>
  </ol>

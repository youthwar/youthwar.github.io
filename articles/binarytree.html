Generic Binary Search Tree Definition

                        ┌────┐          Depth │
 Level 0                │ 23 │                │
                        └──┬─┘                │
                           │                  │
                ┌────┐     │    ┌────┐        │
 Level 1        │ 11 │◄────┴───►│ 32 │        │
                └──┬─┘          └──┬─┘        │
                   │               │          │
           ┌────┐  │      ┌────┐   │   ┌────┐ │
 Level 2   │  8 │◄─┘      │ 13 │◄──┴──►│ 52 │ │
           └────┘         └────┘       └────┘ ▼
            ◄────────────────────────────────►
            Breadth

Above is an accurate representation of a binary search tree.


Theres a few things we can point out the above:
  1. Every value that is less then the parent head is to the left.
  2. Every value greater than the head is to the right.
  3. This tree has a depth of 3 levels
  4. This tree is not balanced because at level 2 there is no value to the right of 8.
